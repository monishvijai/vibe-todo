# My GNOME Extension

A lightweight GNOME Shell extension that adds a custom icon to the top bar.

## Features

- Displays a small icon in the top panel.
- Easy to install and configure.
- Clean and minimal design.

## Installation

1. Clone or download the repository.
2. Place the folder in the GNOME extensions directory:

   ```bash
   ~/.local/share/gnome-shell/extensions/
